# FiveM Add-on Cars

Collection of add-on cars for FiveM.

## Installation

1. Add the [cars] folder to your FiveM resources directory
2. Edit your server.cfg and add the following code

```
start AstonMartinOne77
start AstonMartinVanquish
start AudiA6
```
